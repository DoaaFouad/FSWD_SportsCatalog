Item Catalog Project:

This is a python application that creates a sports catalog website and JSON API. Each sport category displays its items and also provides user authentication using Google. Registered users will have ability to edit and delete their own items.

Software:

Flask
SQLAlchemy

Installation:
VirtualBox
Vagrant
python 2.7

Setting up OAuth
-Go to www.console.developers.google.com
-Sign up and set up google auth api

Run:
-Run python database_config.py to setup the database
-Run python lots_of_items.py to add items to tables
-Run python project.py to run the program
-Open http://localhost:8080/
